# 0.3 版本计划

目标：把个人待办提醒助手从“列表工具”升级成更像日程助手的版本。

状态：已在 `0.3.0` 实现第一版。

## 优先做

1. 日历视图
   - 增加“今天 / 本周 / 月历”视图
   - 点击某一天只看当天事项
   - 每个日期显示待办数量和过期数量

2. 更强时间识别
   - 支持“半小时后”
   - 支持“下周一上午”
   - 支持“每周三下午3点”
   - 支持“每个月1号”
   - 支持“工作日早上9点”

3. 提醒操作
   - 提醒弹出后可以点“完成”
   - 可以点“延后10分钟”
   - 可以点“明天再提醒”
   - 重复任务可以“跳过本次”

## 暂时不做

- 云同步
- 登录账号
- 复杂记账报表
- 多人协作
- 手机后台推送

## 验证标准

- 新建“下周一上午10点开会”后，能出现在正确日期。
- 新建“半小时后提醒我喝水”后，能正确计算时间。
- 新建“每周三下午3点看股票”后，提醒一次后能生成下一次。
- 提醒弹出时，完成、延后、跳过三个操作都能正常更新列表。
